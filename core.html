class core.accountant.Accountant[source]

Bases: object

Monitor and collect data from a running :class:Controller.

The Accountant maintains a dictionary of event:handler mappings. When an event fires in the attached Controller, the corresponding handler collects data.

Attention

One Accountant can only be registered with one Controller; one Controller can have multiple Accountant s.

class core.controller.LinearController[source]

Bases: Controller[T]

A simple evolutionary algorithm.

An evolutionary algorithm that maintains one population and does not take advantage of parallelism. The algorithm applies its operators in the following order:

  1. evaluate for parent selection

  2. parent selection

  3. update population

  4. vary parents

  5. evaluate for survivor selection

  6. survivor selection

  7. update population
